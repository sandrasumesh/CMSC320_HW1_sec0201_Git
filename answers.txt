Q1: Why might we consider using Git?

Answer: Git is useful because it allows users to track changes in their code and work with others without overwriting each other's code. It also allows for quick reversion to original versions of work. 

Q2: What field could data science be applied to? Give an Example.

Answer: Data science could virtually be applied to any field and discipline. Most fields use data to improve decision making and gain insight into important trends and information. One field where this is very important is healthcare, where we could use raw data to notice trends about diseases.
